# CodeActor Skill

将代码库转化为 3D 拟人化可视化剧场。

## 功能描述

CodeActor 是一个创意代码可视化工具，它会：

1. **扫描代码库**：分析项目结构和依赖关系
2. **生成角色人设**：根据模块功能分配性格、外貌和职位
3. **建立社交关系**：将调用关系转化为有趣的社交描述
4. **3D 可视化**：使用 Three.js 程序化生成可爱的卡通角色
5. **健康检测**：识别有 Bug 风险的模块，显示为"生病"状态

## 角色类型

| 性格 | 对应代码 | 特征 |
|------|----------|------|
| 热血主角 | main/app/index | 总是冲在最前面，但容易崩溃 |
| 可靠支柱 | database/model/store | 性格沉稳，记性好但动作慢 |
| 默默助手 | util/helper/common | 深藏不露，总是随叫随到 |
| 古怪角色 | middleware/interceptor | 性格古怪，喜欢设关卡 |
| 神秘人物 | config/constant | 掌握秘密，很少出现 |
| 忙碌蜜蜂 | 高频调用模块 | 总是忙忙碌碌 |
| 脆弱灵魂 | 复杂/易错模块 | 敏感脆弱，需要关怀 |
| 孤独行者 | 无人调用模块 | 独自一人，等待被删除 |

## 关系类型

| 类型 | 代码含义 | 社交描述 |
|------|----------|----------|
| 死党 | 强耦合依赖 | 形影不离 |
| 单恋 | 单向依赖 | 一方依赖另一方 |
| 禁忌之恋 | 循环依赖 | 剪不断理还乱 |
| 匿名树洞 | 异步通知 | 群聊发消息 |
| 偶像崇拜 | 弱依赖 | 粉丝关注 |
| 契约关系 | 接口依赖 | 签订契约 |

## 健康状态

- **健康**：Bug 风险低，绿色显示
- **生病 🤒**：Bug 风险中等，有发热、创可贴效果
- **病重 🚨**：Bug 风险高，有眩晕、警告光环

## 使用方法

### 启动开发服务器

```bash
npm install
npm run dev
```

然后在浏览器中打开 http://localhost:5173

### 作为 Claude Code Skill 使用

直接在 Claude Code 中运行：

```
/code-actor
```

或指定分析路径：

```
/code-actor ./path/to/project
```

## 交互操作

- **拖拽场景**：左键拖拽旋转视角
- **缩放**：滚轮缩放
- **点击角色**：查看详细信息面板
- **拖动角色**：重新布局位置
- **悬停**：显示角色简要信息

## 导出格式

- **JSON**：完整的分析数据
- **Mermaid**：可渲染的关系图
- **叙事文本**：剧场版角色介绍

## 技术栈

- **分析引擎**：TypeScript，支持多语言（优先 JS/TS/Python）
- **3D 渲染**：Three.js + 程序化角色生成
- **前端**：Vite + 原生 TypeScript
